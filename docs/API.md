# API Documentation

## Overview
This document provides detailed information about the API endpoints, request/response formats, and authentication methods.

## Endpoints

### Authentication
- POST /api/auth/login
- POST /api/auth/register
- POST /api/auth/logout

### User Management
- GET /api/users
- GET /api/users/:id
- PUT /api/users/:id
- DELETE /api/users/:id
